# Support & Escalation Model

## Roles
- Operator: runs launcher, monitors basic health.
- AI-IDS Dev Contact: <name>, <email/Discord>
- Chiron Dev Contact: <name>, <email/Discord>

## Where to Report Issues
- Primary: GitHub Issues on this combined repo
  - Labels: bug, ops, doc, enhancement

## Triage Process
1. Operator captures logs, screenshots, exact time.
2. Create GitHub issue with:
   - Steps to reproduce
   - Expected vs actual
   - Relevant logs

## Severity Levels
- Sev 1: System down, no apps usable
- Sev 2: One app partially degraded
- Sev 3: Minor UI bug / cosmetic / docs

## Escalation
- Sev 1: ping both teams immediately
- Sev 2: assigned within 1 day
- Sev 3: added to next sprint
